

# Chikubhai Sarbatwala — Premium Gola Chasani Website

## Overview
A vibrant, visually rich 3-page website for Chikubhai Sarbatwala, a premium gola chasani (flavored syrup) brand with 50+ flavors sold in 6-liter bulk cans. The design will be energetic, summer-inspired, and flavor-forward — bold colors, glassmorphism cards, smooth animations, and strong visual storytelling.

---

## Pages & Features

### 1. Home Page

- **Sticky Navbar** — Brand logo (uploaded chikubhai_logo.jpg), links to Home / Products / Contact, and a WhatsApp CTA button with glow hover effect
- **Hero Section** — Bold animated headline "Always Be Juicy" with subtext "Premium Gola Chasani for Every Flavor", colorful gradient splash background, product can visual, and "View Products" CTA button with glow effect
- **About Section** — Short brand introduction with animated stat counters: 50+ Flavors, 6L Bulk Supply, Years of Trusted Quality
- **Flavor Showcase** — Carousel/grid of 6-8 popular flavors (Mango, Strawberry, Orange, Falsa, Coconut, Chocolate, Kala Khatta, Blue Lagoon) with color-themed gradient cards, hover glow/lift animations, and a "View All Products" button
- **Why Choose Us** — Four glassmorphism feature cards with icons: High Quality Ingredients, Consistent Taste, Bulk Supply Ready, Wide Flavor Range
- **Bulk Packaging Highlight** — Dedicated section emphasizing the 6-liter can format, wholesale/distributor-friendly messaging, and a WhatsApp inquiry CTA
- **Contact Section** — Phone numbers, WhatsApp message button, full address, and embedded Google Maps view for the Ahmedabad location
- **Footer** — Brand name & logo, quick navigation links, contact info, Instagram social link

### 2. Products Page

- **Category Filter Tabs** — Fruit Flavours, Sweet & Dessert, Traditional & Tangy, Special Fusion — with an "All" option
- **Full Flavor Grid** — 80+ product cards, each showing:
  - Flavor name
  - Color-themed gradient background matching the flavor
  - "6L Can" packaging badge
  - Hover animation (lift + color glow)
- **Bulk Order CTA** — Prominent bottom section encouraging wholesale inquiries via WhatsApp

### 3. Contact Us Page

- **Clean business-focused layout** — No form, just direct contact methods
- **Phone numbers & WhatsApp button** — Using wa.me link for instant messaging
- **Full address** with embedded Google Maps for "46, Swaminarayan Industrial Park, Kathwada Shingarwa Road, Ahmedabad, India 382430"

---

## Design & Visual Style

- **Color palette** — Vibrant flavor-inspired gradients used throughout: mango yellow-orange, strawberry pink-red, falsa purple, coconut cream-beige, orange deep-orange, blueberry blue-purple, kiwi green
- **Glassmorphism cards** — Semi-transparent blurred backgrounds for a premium modern feel
- **Typography** — Bold large sans-serif headings for impact; clean readable body text
- **Animations** — Fade-in/slide-up on scroll, hero text reveal, hover glow on buttons and cards, subtle floating decorative drip/splash elements
- **Brand assets** — The uploaded Chikubhai logo used in navbar and footer; product reference photos inform the visual style of can/product imagery throughout

---

## Technical Approach
- Fully static frontend — no backend needed
- All 80+ flavors hardcoded as structured data with category tags and color mappings
- Google Maps iframe embed for location
- WhatsApp links via wa.me API
- Responsive across desktop, tablet, and mobile
- Smooth CSS/JS scroll animations using intersection observer pattern
